## General informations
iTrends script was designed to webscrap information about the number of job offers in IT industry from main Polish portals.
Data is collected in .json file.

## Technologies
* Python 3.8.1
* Beautiful Soup 4 v4.9.0
* Numpy v1.18.1
* Pandas v1.0.1
